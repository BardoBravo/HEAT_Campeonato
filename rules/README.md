# Regras do Campeonato HEAT

Esta pasta deve conter os seguintes arquivos PDF com as regras oficiais:

## Arquivos Necessários:

1. **heat-regras-basicas.pdf** - Regras básicas do jogo HEAT
2. **heat-chuva-forte.pdf** - Regras para condições de chuva forte  
3. **heat-visao-tunel.pdf** - Regras para visão em túnel
4. **heat-lendas-rs.pdf** - Regras específicas para lendas do Rio Grande do Sul

## Como Adicionar:

1. Obtenha os PDFs das regras oficiais do jogo HEAT
2. Renomeie os arquivos exatamente com os nomes listados acima
3. Coloque-os nesta pasta `rules/`
4. Os links no menu lateral direito funcionarão automaticamente

## Observação:

Enquanto os arquivos PDF não estiverem disponíveis, os links no menu abrirão uma página de erro 404, mas isso não afeta o funcionamento do resto da aplicação.
